# Список задач

Человек задаёт порядок и продуктовый замысел. Агенты выполняют только первый
пункт `[ ] ready` и не начинают следующий до завершения или диагностической
остановки текущего. Рабочий PR показывает, что задача выполняется; `[~]` и
блокирующий PR не используется.

Правила цикла: `<methodology-repo>/docs/WORKFLOW.md`.
`BACKLOG.md` — единственный источник истины о задачах. Записи
`.tasks/TASK-NNNN.json` генерируются из него командой
`uv run <methodology-repo>/tools/tasks/sync.py` и вручную не редактируются.

## Выполнено

После слияния и успешного развёртывания агент запускает `tools/tasks/complete.py`:
команда помечает задачу `[x]`, сохраняет агрегированное свидетельство и
синхронизирует `.tasks`. Вручную эти три артефакта не согласовываются.
